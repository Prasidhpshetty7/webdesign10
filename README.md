## 📦 Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (v18 or higher)
- [pnpm](https://pnpm.io/) (recommended) or npm

### Installation

1. **Clone the repository**

```bash
git clone https://github.com/Prasidhpshetty7/portfolio.git
cd portfolio
```

2. **Install dependencies**

```bash
pnpm install
# or
npm install
```

3. **Start development server**

```bash
pnpm dev
# or
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view it in your browser.

4. **Build for production**

```bash
pnpm build
# or
npm run build
```

5. **Preview production build**

```bash
pnpm preview
# or
npm run preview
```

## 🎨 Customization

### Update Personal Information

1. **Site Configuration** - Edit `src/config/siteconfig.json`
2. **About Page** - Edit `src/pages/about-me.vue`
3. **Hero Section** - Edit `src/components/hero/HeroSection.vue`
4. **Social Links** - Edit `src/components/social-media/SocialMediaIcons.vue`
5. **Projects** - Edit `content/projects/projects.json`

### Add Blog Posts

Create markdown files in `content/articles/`:

```markdown
---
title: Your Post Title
date: 2026-03-20
tags: [vue, javascript]
---

# Your Post Title

Your content here...
```

### Environment Variables

Copy `.env.example` to `.env` and add your keys:

```env
VITE_GITHUB_TOKEN=your_github_token
VITE_GOOGLE_ANALYTICS_ID=your_ga_id
VITE_GOOGLE_SEARCH_CONSOLE_TOKEN=your_token
```
